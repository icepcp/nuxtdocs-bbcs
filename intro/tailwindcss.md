# What’s TailwindCSS?
Tailwind CSS is a popular CSS framework that provides a set of pre-designed utility classes to help developers rapidly build user interfaces.
## Code comparison with and without TailwindCSS
### With TailwindCSS
```html
<nav class=”max-w-xl”>
	<!-- nav -->
</nav>
```
### Wthout TailwindCSS
```html
<nav class=”container”>
	<!-- nav -->
</nav>
```
```css
.container {
  max-width: 36rem;
}
```
